Introduction
============

This package won't be usable with the current out-of-the-box Plone
installations.

The experiment was done using:

* http://http://dist.plone.org/release/4.0.4/versions.cfg
* http://good-py.appspot.com/release/dexterity/1.0b7?plone=4.0.4
* http://good-py.appspot.com/release/plone.app.z3cform/0.5.3?plone=4.0.4

With:

* z3c.form = 2.4.1
* plone.autoform = 1.0b4

* plone.app.blocks = 1.0a1dev-r47518
* plone.app.dexterity = 1.0b3dev-r47466
* plone.app.drafts = 1.0a1dev-r47152
* plone.app.tiles = 1.0a1dev-r47546
* plone.dexterity = 1.0b7dev-r47454
* plone.directives.tiles = 1.0a1dev-r39977
* plone.resource = 1.0dev-r47271
